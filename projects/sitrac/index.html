SITRAC → SIIGO + DIAN

Automatización de un flujo operativo completo: registro de egresos, creación de documentos en SIIGO y pasos de gestión tributaria (DIAN), integrando validaciones, reintentos y trazabilidad del proceso.

Python Selenium / SeleniumBase Automatización end-to-end Integración de procesos Excel (lotes)

Qué problema resuelve

  • Reduce trabajo manual repetitivo entre sistemas sin integración directa.
  • Disminuye errores por digitación y mejora consistencia del proceso.
  • Acelera el flujo operativo con validaciones y control de estados.
  • Permite trazabilidad (qué se hizo, cuándo y con qué resultado).

Qué hace

  • Lee una entrada (ej. archivo o listado) y ejecuta el flujo por cada registro.
  • Automatiza tareas web (login, navegación, formularios, confirmaciones).
  • Registra resultados del proceso (estado OK / error / pendiente) y evidencia.
  • Incluye manejo de errores y reintentos para estabilidad.

Flujo resumido

Entrada (lote) → Egresos (SITRAC) → Registro/creación (SIIGO) → Validación → Estado final / evidencia.

Tipo RPA / Automatización web con Python
Alcance Proceso end-to-end (varios sistemas)
Robustez Esperas, reintentos y manejo de errores
Salida Estados, trazabilidad y evidencia
Nota de confidencialidad: por seguridad y acuerdos de trabajo, no se publica código, credenciales ni datos de terceros. La demostración se enfoca en el flujo y el resultado.
